# Validation

- `schema.json`: machine-readable shape for a validation manifest when a new case is proposed.
- `analytical_beam/`: committed analytical comparison for the cantilever beam workflow.
- `public_formula_checks/`: committed hand-calculation comparisons against public formulas.
- `roark_formulas/`: sourced square-plate benchmark for the clamped analytical estimator.
- `mesh_convergence/`: committed Docker-backed cantilever beam convergence study.
- `solver_comparison/`: intake scaffold for same-prompt cross-backend comparisons.
