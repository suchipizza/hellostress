# Examples

- `minimal/`: fastest supported end-to-end run.
- `beam_cantilever/`: supported beam example with expected metrics.
- `simply_supported_beam/`: supported analytical beam example with a roller boundary condition.
- `plate_pressure/`: supported rectangular-plate pressure example.
- `hand_calc_comparison/`: supported beam example with explicit analytical derivation.
- `plate_with_hole/`: analytical mock-mode tension screening for a plate with a central hole.
- `bracket_linear_elastic/`: analytical mock-mode bracket screening example.

Run the smoke checks with:

```bash
./examples/smoke_test.sh
```
