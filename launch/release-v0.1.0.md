# v0.1.0 — Open-source baseline

## Highlights

- public README and quickstart for FEA Copilot
- committed examples for beam, plate, simply supported beam, bracket screening, and plate-with-hole screening
- validation coverage for analytical beam, hand calculation, Roark-style plate comparison, and mesh convergence evidence
- CI, docs, contributor scaffolds, launch drafts, and devcontainer setup

## Notes

- bracket and plate-with-hole workflows are analytical `mock`-mode screening only
- Docker execution is currently scoped to beam and rectangular-plate workflows
- this release is intended as the public baseline for future benchmark and education work
